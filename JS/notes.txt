variables:
------------
    ->name and memory location used to store data or values and it can change N number of times during the execution

    1. variable declaration (defining variable):
        ex: let a=10; let a=20;
    2.variable initialization :
        ex: a=10;
    3. variable utilization 
    4. variable re-initialization
    5. variable re-declaration

In JS we can define variable in 3 ways:

    1. let:
        ->acts as local scope/ block scope
        ->declaration and initialization in the same/different line is possible
        ->re-initialization is possible but re-declaration is not possible
        ->we can't hosit the let variable
            hositing:
            -----------
             ->utilizating the variables before declaration
             -> only possible throught 'var' variable
     2.var
           ->let variables acts like a global scop/block scop (because var present in windows object)
           ->variable declaration and initialization in  same/different line is possible
           ->re-initialization and re-declaration is possible
           ->we can hoist the var variables
    3.const
           ->const variables acts like like a final (because once initialization values we cant modify data/values)
           ->variable declaration and initialization in  same is possible but different lines is not possible
           ->re-initialization and re-declaration is not possible
           ->we cant hoist the const variable
